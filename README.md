# Rust Programming Language Studies

This repository is dedicated to exploring and testing the concepts, guidelines, and examples provided in **[The Rust Programming Language](https://doc.rust-lang.org/stable/book/)** (often referred to as "The Book").

## Current Projects

- **hello_cargo**: An introduction to using Cargo, Rust's build system and package manager.
- **hello_everyone**: A variation of the classic "Hello, World!" project.
- **hello_world**: Basic Rust entry point.
- **guessing_game**: A simple interactive console game to learn about variables, input, and matching.

## Purpose

The main focus is to:
1.  Implement code examples from official documentation.
2.  Experiment with Rust syntax and ownership rules.
3.  Practice project organization using Cargo.

## Getting Started

Ensure you have Rust and Cargo installed. You can install them via [rustup](https://rustup.rs/):

```powershell
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

To run any of the projects:

```powershell
cd <project_folder>
cargo run
```
